¿Qué es un GIF?

Es un especialista académico y/o operativo encargado de analizar, corregir y proponer soluciones innovadoras para la creación de ambientes virtuales de aprendizaje, a partir de contenidos educativos.

01. Recepción del Requerimiento

Envío por correo a: fabricadecontenidos@cun.edu.co

Datos requeridos: asignatura, créditos, syllabus, docente

02. Validación del Syllabus

Verificación por parte del Especialista GIF para asegurar cumplimiento del modelo virtual.

03. Asignación de GIF Responsable

El GIF coordina todo el proceso de virtualización y actores involucrados.

04. Solicitud de Experto Temático (si aplica)

Formulario de disponibilidad presupuestal.

05. Inicio de Virtualización

Con acompañamiento del GIF, se inicia la construcción de contenidos.

06. Trámite de Cuenta de Cobro

Documentos: contrato, actas, cuenta de cobro, no declarante

¿Proceso de Virtualización?

El proceso de virtualización educativa capacita a los docentes en el uso de herramientas como ChatGPT, Gamma y Fliki para crear contenidos interactivos. Se enfoca en metodologías activas, con objetivos claros y evaluaciones como quizzes y ejercicios prácticos. La virtualización se certifica institucionalmente, garantizando calidad académica.

¿Proceso de Seguimiento Docente?

El proceso de seguimiento docente permite garantizar el cumplimiento de los contenidos y la calidad educativa. A través de herramientas de control y retroalimentación, se monitorean las entregas, el avance del plan de trabajo y la aplicación de metodologías. Este proceso fortalece la experiencia de aprendizaje virtual.
